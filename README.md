# Todo

> Todo 可以帮助你计划、管理、执行、记录工作。支持多种终端设备，包括PC、iPad、Phone等。

## 安装

``` bash
# 安装依赖
npm install

# 启动服务
npm run start

# 建立可发布文件
npm run build

```
